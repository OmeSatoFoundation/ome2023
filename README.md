# 子ども IT 未来塾講義用マテリアル
子供向け IT 教育用の教材ソースです。
Raspberry Pi を使うことを前提とした演習環境と講義用教科書・プレゼンテーションスライドが含まれます。

## How to Use
本リポジトリは Raspberry Pi OS を基にした演習環境と
lualatex で執筆された教科書・プレゼンテーションスライドからなります。
それぞれの詳細な使用方法は

- [docs/README_os.md](docs/README_os.md)
- [docs/README_text.md](docs/README_text.md)

をご確認ください。

## CONTRIBUTING
本プロジェクトに貢献する際や、仕様とポリシーの確認には [docs/CONTRIBUTING.md](docs/CONTRIBUTING.md) をご確認ください。
